Công việc theo nhóm

Soạn tin · gửi Zalo · theo dõi trạng thái

Bộ lọc nâng cao

Hướng dẫn gửi tin (Zalo PC / Web)

App Zalo PC: Gửi tin trên app → đổi trạng thái sang Đã gửi trên thẻ nhóm.
Gửi (Web): Tab chat.zalo.me mở sẵn → bấm 📤 Gửi (Web) — gửi ngầm.

Thao tác hàng loạt

Bạn cần làm 3 bước (theo thứ tự)

  1. Import nhóm từ Zalo — bấm nút cam bên dưới
  2. Tạo chiến dịch — cột phải → đặt tên → Tạo
  3. Menu Công việc → chọn chiến dịch → soạn tin → Gửi Web

Danh sách nhóm Zalo

+ Thêm nhóm thủ công (ít dùng)

Toàn bộ trong web — không cài Tampermonkey. Admin: quét Zalo Web hoặc dán/CSV → import một lần.

① Quét nhóm tự động (khuyên dùng — 500+ nhóm)

  1. CRM → Đồng bộ Zalo → Tạo mã đồng bộ → Copy mã
  2. Chrome cài extension: tools/zalo-sync-extension (Load unpacked)
  3. Mở chat.zalo.me → dán mã → bấm Quét nhóm → gửi CRM (extension tự cuộn & quét)
  4. Quay CRM → bấm Hiện nhóm vừa quét từ Zalo bên dưới
Chỉ import:

①b Hoặc bookmark (dự phòng)

  1. Kéo nút vào thanh dấu trang (bookmark) — chỉ làm một lần: Quét nhóm Zalo → copy CSV
  2. Mở chat.zalo.me, đăng nhập, cuộn danh sách chat/nhóm bên trái.
  3. Bấm bookmark vừa kéo → CSV được copy vào clipboard.

② Dán hoặc chọn file CSV

Mỗi dòng: Tên nhóm hoặc CSV ten_nhom, nguoi_phu_trach, zalo_group_id — Tải file mẫu
