Hướng dẫn sử dụng Zalo Campaign CRM

Tài liệu dành cho team vận hành: quản lý ~300 nhóm Zalo, mỗi nhóm một nội dung riêng, theo dõi đã gửi / follow-up, thỉnh thoảng gửi thông báo chung nhiều nhóm.

CRM làm gì: Soạn tin, phân việc, theo dõi trạng thái, báo cáo.
CRM không làm gì: Không thay Zalo, không tự gửi 1000 tin/ngày kiểu bot (tin vẫn do người gửi trên Zalo Web hoặc app Zalo).

Mục lục

  1. Chuẩn bị & đăng nhập
  2. Cài extension Chrome
  3. Quy trình 3 bước (bắt buộc đọc)
  4. Từng menu trong CRM
  5. Gửi tin hàng ngày (Công việc)
  6. Thông báo hàng loạt

1. Chuẩn bị & đăng nhập

Chạy CRM trên máy

  1. Mở terminal trong thư mục zalo-crm-mvp
  2. Chạy: npm.cmd start
  3. Mở trình duyệt: http://localhost:3000/login.html

Tài khoản demo

Vai trò User Mật khẩu
Admin (full quyền) admin admin123
Người soạn tin editor editor123
Người trả lời responder responder123
Trên Windows nếu lỗi npm, dùng npm.cmd thay cho npm.

2. Cài extension Chrome (một lần)

  1. Chrome → Extensions → bật Developer mode → Load unpacked → chọn thư mục tools/zalo-sync-extension
  2. CRM → menu Đồng bộ Zalo → bấm Tạo / làm mới mã đồng bộ (Admin) → F5 trang CRM
  3. Mở tab chat.zalo.me → đăng nhập Zalo → giữ tab này mở khi dùng Gửi Web
  4. Sau khi cập nhật extension, bấm Reload extension rồi F5 CRM + F5 chat.zalo.me

3. Quy trình 3 bước (làm theo thứ tự)

Bước 1 — Import nhóm (menu Nhóm & chiến dịch)
Extension quét Zalo → CRM import danh sách nhóm vào hệ thống.
Bước 2 — Tạo chiến dịch (cùng trang, cột phải)
Mỗi đợt gửi tin = một chiến dịch (VD: «Tháng 5 — chăm sóc khách»).
Bước 3 — Soạn & gửi (menu Công việc)
Chọn chiến dịch → soạn tin từng nhóm → Gửi Web hoặc gửi tay trên Zalo → đổi trạng thái Đã gửi.

Import nhóm từ Zalo (chi tiết)

  1. Menu Nhóm & chiến dịch → bấm nút xanh Import nhóm từ Zalo / CSV
  2. Trên chat.zalo.me: extension popup → Quét nhóm → gửi CRM (đợi 1–3 phút nếu nhiều nhóm)
  3. Quay CRM → Hiện danh sách vừa quét → chọn Chỉ import: Nhóm Zalo
  4. Bấm Import vào CRM
Tìm nhóm: Dùng ô Tìm tên nhóm, người phụ trách… (viền xanh).
Không dùng ô «Thêm nhóm thủ công» để tìm — ô đó chỉ để thêm nhóm mới bằng tay.

4. Từng menu trong CRM

Menu Dùng khi nào
Công việc Hàng ngày: soạn tin, Gửi Web, đổi trạng thái, follow-up
Nhóm & chiến dịch Import nhóm, tạo chiến dịch, tìm nhóm
Tổng quan Xem số liệu: chưa gửi, đã gửi, quá hạn follow-up
Thông báo Thỉnh thoảng: 1 nội dung gửi nhiều nhóm (không giới hạn ~100 nhóm)
Đồng bộ Zalo Tạo mã sync, kiểm tra extension online/offline
Inbox Xem nhanh nhóm theo SĐT / hồ sơ (nâng cao)
Cài đặt CRM Template tin, webhook, cấu hình nâng cao

5. Gửi tin hàng ngày (menu Công việc)

Trạng thái từng nhóm

Trạng thái Ý nghĩa
Chưa gửi Chưa gửi tin trong chiến dịch này
Đã gửi Đã gửi tin lên Zalo
Đang trả lời Khách đã phản hồi, đang xử lý
Hoàn tất Xong việc với nhóm này

Cách gửi

  1. Chọn chiến dịch ở dropdown trên cùng
  2. Mở thẻ nhóm → soạn nội dung trong ô tin nhắn
  3. Gửi Web — gửi ngầm qua tab chat.zalo.me (bạn ở lại CRM). Cần: extension + mã sync + đăng nhập Zalo Web
  4. Hoặc Mở Zalo → gửi tay → đổi dropdown trạng thái sang Đã gửi
Nếu Gửi Web báo «Không tìm thấy nhóm»: trên chat.zalo.me click đúng nhóm bên trái một lần, rồi thử lại. Tên nhóm trong CRM nên khớp tên trên Zalo.

Lọc & tìm kiếm (Công việc)

  • Ô tìm kiếm trên cùng — lọc theo tên nhóm, nội dung, người phụ trách
  • Bộ lọc trạng thái, ưu tiên, follow-up quá hạn
  • Phân trang 10 / 25 / 50 / 100 nhóm mỗi trang
  • Export CSV — xuất báo cáo theo bộ lọc hiện tại

6. Thông báo hàng loạt (menu Thông báo)

Dùng khi cần cùng 1 nội dung gửi nhiều nhóm (VD: thông báo nghỉ lễ). Ít dùng hơn tin riêng từng nhóm.

  1. Tạo thông báo mới → tiêu đề + nội dung chung
  2. Chọn nhiều nhóm (không giới hạn ~100 như app Zalo)
  3. Lưu → chọn thông báo ở menu «Đang xem»
  4. Gửi từng nhóm bằng Gửi Web hoặc chọn nhiều → Gửi Web (đã chọn)
  5. CRM theo dõi nhóm nào Đã gửi / chưa gửi

7. File đính kèm (ảnh, PDF, Excel, XML…)

  • Trên thẻ nhóm → Chi tiết → + Thêm file
  • File lưu trên CRM để team biết cần gửi kèm gì
  • Gửi file vẫn làm tay trên Zalo (Web hoặc app) — CRM không tự đính kèm file lên Zalo

8. Phân quyền

Vai trò Được làm Không được
Admin Full: import, xóa, mã sync, cài đặt —
Editor Soạn tin, gửi Web, tạo broadcast Xóa nhóm/chiến dịch
Responder Đổi trạng thái (đã gửi, đang trả lời…) Sửa nội dung tin, import

9. Lỗi thường gặp

Search / tìm nhóm không ra

  • Dùng ô Tìm… viền xanh (menu Nhóm & chiến dịch), không dùng ô «Thêm nhóm»
  • Thử bấm filter Tất cả thay vì chỉ «Nhóm»

Gửi Web không được

  • CRM → Đồng bộ Zalo → có mã sync chưa?
  • Tab chat.zalo.me mở và đã đăng nhập?
  • Reload extension v1.7+ → F5 CRM → F5 Zalo
  • Đã chọn chiến dịch và soạn nội dung tin chưa?

Quét nhóm thiếu

  • Đợi extension quét xong (nhiều nhóm = vài phút)
  • Cuộn danh sách chat bên trái trên Zalo trước khi quét
  • Quét lại → import merge (không xóa nhóm cũ)

Server không chạy

  • npm.cmd start trong thư mục dự án
  • Cần PostgreSQL đang chạy (Docker hoặc cài local)

10. Checklist kiểm tra nhanh

  • Đăng nhập admin OK
  • Extension cài + mã sync tạo xong
  • Quét Zalo → import ≥ 1 nhóm
  • Tạo 1 chiến dịch
  • Công việc: soạn tin → Gửi Web → trạng thái Đã gửi
  • (Tuỳ chọn) Thông báo hàng loạt 2–3 nhóm
  • (Tuỳ chọn) Thêm file đính kèm trên thẻ nhóm
